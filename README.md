dotfiles
========

Prerequisites
=============
1. VirtualBox  - https://www.virtualbox.org/wiki/Downloads
2. Vagrant - http://downloads.vagrantup.com/

Installation
============
1. Clone this repository

    git clone git@github.com:mattcaldwell/dotfiles.git

2. Launch vagrant VM

    cd dotfiles
    vagrant up

Using
=====
1. vagrant ssh
2. and you're done!
